# SalesDashboard 
## Overview This project demonstrates how to query sales data using SQL, export the results to CSV, and visualize them in Power BI. The dashboard highlights sales performance by salesperson and territory, along with a key KPI showing total sales.

## Files 
`Sales_by_Person.sql` → SQL query script 
 `Sales_by_Person.csv` → Query results dataset 
 `SalesDashboard.pbix` → Power BI project file 
 `SalesDashboard.png` → Dashboard visualization
 
 ## Dashboard The dashboard contains: 
 **Bar chart**: Total sales by salesperson 
 **Stacked bar chart**: Sales by salesperson broken down by territory 
 **Card visualization**: Total sales KPI (78.49M)
